# leaf-disease
# Project Description 
Plant diseases can significantly reduce crop yield and quality. Early detection is essential for effective treatment and improved productivity. This project uses a **Convolutional Neural Network (CNN)** model trained on leaf image datasets to classify healthy and diseased leaves. Users upload a leaf image through the web interface, and the application predicts the disease in real time.
# Features
Upload leaf images,
AI-powered disease detection,
High prediction accuracy,
User-friendly Flask web application,
